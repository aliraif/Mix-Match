# Mix & Match

## Tetris Game 
Project Description : A modern recreation of the classic Tetris game featuring smooth block movement, rotation, line clearing, and scoring. The game includes Classic, Endless, and Career Modes, enhanced with responsive UI, sound effects, and background music. Players place falling shapes to clear lines and prevent the board from filling up
